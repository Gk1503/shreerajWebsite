# Catalogues Folder

This folder contains PDF catalogues for each category.

## File Naming Convention

For each category, you need TWO PDF files:

1. **View PDF** (without watermark) - for online viewing
2. **Download PDF** (with watermark) - for downloads

### Naming Format:
- View PDF: `category-[number]-view.pdf`
- Download PDF: `category-[number]-download.pdf`

## Example for Category 1:

1. Place your PDF without watermark here and name it:
   `category-1-view.pdf`

2. Place your PDF with watermark here and name it:
   `category-1-download.pdf`

## How to Upload:

1. Copy your PDF files to this folder: `frontend/public/catalogues/`
2. Rename them according to the naming convention above
3. The website will automatically use these files

## Important Notes:

- File names are case-sensitive
- Use lowercase for file names
- Make sure both view and download versions exist for each category
- View version = NO watermark (for online preview)
- Download version = WITH watermark (for downloads)
